Nginx
קובץ ראשי config/nginx/mydjango.conf 
client_max_body_size 10M;

        upstream web {  
          ip_hash;
          server web:8001;
        }
        
        server {
        
            location /static/ {    
                autoindex on;    
                alias /src/static/; 
            }
        
            location /media/ {
                autoindex on;
                alias /src/media/;
            }
        
            location / {
                proxy_pass http://web/;
            }
            listen 8001;
            server_name localhost;
        }
        
חובה לרענן מכולת SiteWeb לאחר שינוי קוד פייטון!

פקודות חשובות:

docker-compose build בניית מערכת

docker-compose up הפעלת מערכת + לוגים
